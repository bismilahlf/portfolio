R-Tail
El e-commerce de moda que se adapta a cada usuario

R-Tail está enfocada a ofrecer productos de alta calidad y una experiencia de compra personalizada.

User Research

Según los resultados de nuestras entrevistas, una de las mayores preocupaciones de nuestros usuarios cuando compran ropa online es acertar con la talla. Además, lo que más valoran de los comercios locales es que "conozcan sus gustos".

FEATURES

Algoritmo de recomendación

La experiencia de cada usuario cambia a partir de sus compras y gustos.

FEATURES

Calculador de tallas

El proceso de registro permite a los usuarios introducir sus medidas, de manera que toda la experiencia de compra se ajuste a ellos.

FEATURES

Fotografía inclusiva

Cada prenda está fotografiada en modelos con distintos tipos de cuerpo, de forma que al usuario le resulte más fácil identificarse con ellos.
